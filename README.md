# Crater - Invoice & Billing System

## 📌 Overview
Crater is an open-source invoicing and accounting application that helps businesses manage invoices, payments, and expenses efficiently.

## 🚀 Features
- Invoice Creation & Management
- Payment Tracking
- Expense Management
- Customer & Vendor Management
- Multi-currency Support
- Dashboard Analytics
- Tax Calculation

## 🛠️ Tech Stack
- Backend: Laravel
- Frontend: Vue.js
- Database: MySQL
- API: RESTful API

## ⚙️ Installation
1. Clone repository:
   ```bash
   git clone https://github.com/crater-invoice/crater.git

Navigate:

cd crater

Install dependencies:

composer install
npm install

Configure environment:

cp .env.example .env
php artisan key:generate

Run migrations:

php artisan migrate

Start app:

php artisan serve
